# Quanvio Public Assets

Public static assets for Quanvio products. These files are served via `raw.githubusercontent.com`.

## Structure

```
img/
  qnx-studio/        - QNX Studio VS Code extension images
    icon.png         - Extension marketplace icon (128x128)
    sidebar-icon.png - Activity bar icon (24x24, white on transparent)
    readme.png       - Extension Details banner
    logo-dark.svg    - QNX logo for dark backgrounds
    logo-light.svg   - QNX logo for light backgrounds
sounds/              - Future audio assets
```
